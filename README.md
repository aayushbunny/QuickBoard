# QuickBoard 

QuickBoard is a modern Android ticket booking application built with Kotlin and Jetpack Compose. It enables users to book local train and bus tickets, make secure payments, and access digital QR-based tickets.

## ✨ Features

* Local Train Ticket Booking
* Bus Ticket Booking
* Firebase Authentication
* Firestore Database Integration
* Razorpay Payment Gateway
* QR Code Based Digital Tickets
* My Bookings Screen
* Modern Material 3 UI
* Real-time Booking Management

## 🛠️ Tech Stack

* Kotlin
* Jetpack Compose
* Material 3
* Firebase Authentication
* Cloud Firestore
* Razorpay
* Navigation Compose
* MVVM Architecture

## 🚀 Getting Started

### Prerequisites

* Android Studio
* JDK 17+
* Firebase Project
* Razorpay Account

### Installation

1. Clone the repository

```bash
git clone https://github.com/aayushbunny/QuickBoard.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Add your Firebase configuration

5. Run the application

## 📂 Project Structure

* `app/` - Android application source code
* `functions/` - Firebase Cloud Functions
* `gradle/` - Gradle configuration files
